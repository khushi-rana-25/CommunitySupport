Notification Popup Component

Dark theme notification popup with customizable content

Usage Example:

// Show notification with custom content showNotification('info', 'Your custom message here', true, true); // Parameters: // 1. type: 'success', 'warning', 'info', 'error' // 2. body: Your custom message text // 3. showCancel: true/false (show cancel button) // 4. showOkay: true/false (show okay button) // Example usage in your pages: showNotification('success', 'Profile updated successfully!', false, true); showNotification('warning', 'Are you sure you want to logout?', true, true);
